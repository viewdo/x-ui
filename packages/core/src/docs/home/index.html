Modern web-application functionality without leaving the comfort of HTML.

Turbo-charged HTML

Add multi-page routing, guided navigation, data-binding, dynamic content, and even event-based actions to any HTML file with little scripting and no framework.

This documentation was created with these experience elements. View-source or take the tour to see how complex routing, guided navigation, automatic presentations, data-binding and more is made possible!

<x-ui/> is a suite of custom HTML elements that enable non-developers to turn static HTML files into single-page web-applications, with many features to make rich web-based experiences, easily.

Not every web-page needs a web-framework.

Frameworks minimize the effort to efficiently create and maintain web pages.

However, not everyone has the coding skills, framework expertise, or the patience for build systems, app servers, etc.

No Framework, No Problem

<x-ui/> is different. Without transpilation, a framework or even a build-process, designers and developers get the same time-saving techniques (layouts, routing, data-binding, etc) using specialized elements directly in the HTML.

Nothing to clear with IT

YES. If you can include a <script> tag in the header, you can turn your static HTML into a full-fledged web application. Host it on your favorite static server -- there is no run-time and no server dependencies.

What it is not.

<x-ui/> experience elements add functionality, NOT design. These elements work with your design and since it is just HTML, you can use it with any component or UI library you want.

Features:

  • Application-like Routes w/Lazy Loaded HTML
  • Support for Data Routes for Data-Driven Content
  • Use Route Data Directly in HTML w/Route and URL Data Providers
  • Automatic Back & Forward Resolution
  • Sequenced Content: present, educate, inform
  • Sequenced Input: surveys, quiz and display results
  • Sequenced Video: interactive video story-telling
  • Declarative: Expression-based Conditions
  • Stateful: pick up where a user left-off
  • Data Tags to render personalized data from a variety of sources
  • Use HTML to access data from your API
  • Capture attention and data, with smart workflows to replace boring forms.
  • Personalize the content for each of your visitors.
  • Interactive Videos
  • Queued, Timed & Event-based Actions
  • Animated Transitions
  • Background Music & Voice-Overs
  • Animation Hooks
