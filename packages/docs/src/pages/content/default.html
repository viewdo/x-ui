How it works

Include HTML components, fragments or shared widgets with ease using our content components. Just specify the source location and the element does everything else.

Use Markdown to

Pull in markdown content and have it converted to HTML on the fly. This makes writing content easy without worrying about formatting or complicated styles. Just use the <x-content-markdown> component.

On Demand, Non Duplicated References

Once you start breaking your site up into fragments, you'll find it improves performance to not pull in any remote scripts or style-sheets until you need them. Using the <x-content-reference> component, you decorate fragments with the remote dependencies that HTML needs, and the component will make sure it's there. If the user already went to a page-title with that component, it won't be added again. Keep your HTML light and functional.
