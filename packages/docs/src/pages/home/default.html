What is it?

<x-ui/> is a set of custom HTML elements (web components). They enrich static HTML files with features typically only available with JavaScript framework. Our focus is on creating experiential websites without script or a build-process.

What it is not.

<x-ui/> experience elements add functionality, NOT design. These elements work with your design. Since it is HTML, you can always use it with any framework, component library or UI kit you want.

No Framework Necessary

The elements bring time-saving techniques like shared-layouts, reusable widgets, URL-based routing, lazy-loading, data-binding, rule-based workflows, way-finding, markdown, and much more, directly in HTML.
